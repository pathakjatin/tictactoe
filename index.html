Welcome to Jatin's Tic Tac Toe



Rules for playing:

  1. Game is played on a 3x3 grid

  2. If you are X then the opponent is O

  3. Player takes turns putting their marks in empty squares

  4. The first player to get 3 of his/her marks in a row (up,down,across) wins


Enjoy Playing!
